# MVP-002 Result Page

## Goal
Present the result in a visual-first, beginner-friendly way.

## Required hierarchy
1. dominant hero visual
2. main issue
3. fix cue
4. drill suggestion
5. key moments
6. low-weight metadata

## Voice
- plain English
- short
- confidence-building
- not coach-jargon heavy

## Do not include in MVP
- multiple issue cards
- ranking tables
- angle charts
- biomechanical metric grids
